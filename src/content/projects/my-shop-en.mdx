## What it's about

A complete web shop as a React SPA, built in the "Web Programming with React" course at Furtwangen University. The frontend talks to a REST backend (provided by the course) and covers the typical shop flow: browse articles, add to cart, order.

## Features

- **Article catalog with search** — article cards with live search across the inventory
- **Shopping cart** with an item counter right in the app bar
- **Internationalization** via i18n with dedicated locale files
- **Theming** with a centrally managed theme
- **Central store** for application state (cart, user, UI state)
- API layer cleanly separated from the components

## What I learned

Slicing components is the real craft in React — when does something deserve to be its own component, when should state be lifted, when does it belong in the store? Also: designing an internationalized UI from the start is much cheaper than retrofitting it. The experience from this project went straight into this website.
